# Cluster Toolkit

## Overview
Open-source toolkit for creating HPC and AI/ML clusters on Google Cloud. Simplifies deployment of compute clusters for scientific computing and machine learning workloads.

## Repository Purpose
- **HPC cluster deployment** - High-performance computing on GCP
- **AI/ML infrastructure** - GPU/TPU cluster setup
- **Terraform-based** - Infrastructure as code
- **Best practices** - Optimized cluster configurations

## Use Cases
- Scientific computing clusters
- AI/ML training infrastructure
- Batch processing workloads
- Research computing

## Target Audience
- HPC administrators
- ML engineers
- Research institutions
- Cloud architects
